Inference API

Check out the SWAGGER UI for examples on how to use the API and its available options created by Ryan Young version 0.1

Basic GET Request

curl -X 'GET' \
    'http://127.0.0.1:5000/api/enrich?src=https://i.imgur.com/4zSAWJ5.jpeg&model=ResNet&format=image' \
    -H 'accept: */*'

Basic POST Request

curl -X 'POST' \
    'http://127.0.0.1:5000/api/enrich' \
    -H 'accept: */*' \
    -H 'Content-Type: application/json' \
    -d '{
    "src": [
        "https://i.imgur.com/4zSAWJ5.jpeg"
    ],
    "model": "ResNet",
    "format": "image"
    }'
